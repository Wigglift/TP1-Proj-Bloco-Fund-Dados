# 📋 Sistema de Lista de Tarefas (CLI)

Tarefa desenvolvido para a disciplina de **Projeto de Fundamentos de Dados**, com o objetivo de aplicar conceitos de programação orientada a objetos, manipulação de dados e interação com o usuário via terminal.

---

## 🚀 Sobre o Projeto

Este sistema permite o gerenciamento de tarefas através do terminal, possibilitando ao usuário:

- Adicionar novas tarefas
- Listar todas as tarefas cadastradas
- Marcar tarefas como concluídas
- Remover tarefas
- Sair do sistema

As tarefas possuem informações como descrição, data de criação, prazo final, nível de urgência e status.

---

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Manipulação de datas com `datetime`
- Estruturas de controle (`match`, loops)
- Entrada e saída de dados via terminal
- Organização de código em classes

---

## ⚙️ Funcionalidades

### ➕ Adicionar tarefa
Permite criar uma nova tarefa informando:
- Descrição
- Prazo final
- Nível de urgência

---

### 📄 Listar tarefas
Exibe todas as tarefas cadastradas com seus detalhes.

---

### ✅ Marcar como concluída
Atualiza o status de uma tarefa para "concluída".

---

### ❌ Remover tarefa
Remove uma tarefa da lista com base no seu ID.

---

### 🚪 Sair
Encerra a execução do programa.

---
